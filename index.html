JustMail

JustMail is a simple mail server written in Java

View on GitHub Download .zip Download .tar.gz

In Development

The mail server is currently in development and is not yet ready for an usage in production. You can see progress on the development branch. You're welcome to contribute and help to improve everything.

Progress

This is the progress of the server. Things written in italics won't be in the first release, whereas bold things are finished!

  • SMTP-Protocol
  • Command for the console
  • IMAP-Protocol
  • Junk detection
  • Webinterface

Release

A date is currently not specified, but I target somewhere in October. The first release will be usable, but it won't include all planned features.
